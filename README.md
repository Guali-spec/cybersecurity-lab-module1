## Project Description

This project documents a hands-on cybersecurity laboratory aimed at identifying fundamental security weaknesses in everyday computing. Using Wireshark to capture and analyze real network traffic, the lab explores the risks associated with weak passwords, unencrypted communications, and exposed metadata, while demonstrating the protective role of modern encryption protocols such as TLS. The analysis provides practical insight into how data travels across networks and why secure practices are essential for maintaining privacy and system integrity.
